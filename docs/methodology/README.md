## Locked model

**Football IQ:** Awareness, Game Reading, Decision Quality, Adaptability and Use of Space.

**Match Mentality:** Persistence, Recovery, Learning, Composure and Consistency.

**Educational progression:** Notice → Understand → Choose → Adapt → Move → Perform.

**North Star:** PitchIQ develops independent football thinkers through evidence-informed assessment, purposeful practice and meaningful coaching.

Changes to locked methodology require an explicit, versioned approval rather than silent edits.

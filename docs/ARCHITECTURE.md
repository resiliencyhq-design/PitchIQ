# PitchIQ Integrated Architecture v1.0

This repository uses modular static JavaScript for GitHub Pages deployment.

Folders:
- js/app: route rendering and orchestration
- js/game: XP/progression/scoring
- js/services: camera, voice, storage
- js/components: shared UI utilities
- js/data: cues and rewards
- css: design tokens and interface
- assets: brand, artwork and icons

Designed to be migrated to React/Vite later without changing core services.

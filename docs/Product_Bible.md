# PitchIQ Product Bible

## Vision
Train faster. Think faster. Become a better footballer.

PitchIQ is a youth football cognitive performance app. It should feel like a premium academy experience, not a generic dashboard or arcade game.

## Product Principles
- Player is inside the drill, not browsing menus.
- One primary action per screen.
- Consistency beats novelty.
- Behavioural development over UI complexity.
- Psychological safety: reset, recover, continue.

## Design Principles
- Dark premium academy atmosphere.
- Lime accent used sparingly for action and progress.
- Large, confident typography.
- Low UI density.
- No screen should feel like a generic admin dashboard.

## Component Principles
- Hero layouts must separate identity, visual focus, and key stats.
- Cards must be consistent, readable, and reusable.
- Primary actions must be visually decisive.
- Bottom navigation must remain a single app-shell element.

## Codex Compliance Rule
“Codex is an implementation engine, not a designer. If a requested change conflicts with the Product Bible, Codex must stop and explain the conflict rather than inventing an alternative.”

# PitchIQ Design Studio

## Purpose
The Design Studio is the source of truth for UI decisions.

## Design Rules
- Use the Product Bible first.
- Use layout system primitives before ad-hoc CSS.
- Mobile-first at 390–430px.
- No horizontal scrolling.
- No overlapping structural zones.
- One primary action per screen.

## Component Library
- HeroLayout
- CardBase
- StatCard
- MissionCard
- QuickTile
- ActionButton
- BottomTabBar
- TrainingCue
- FeedbackState

## Colour System
Dark stadium base, glass surfaces, lime accent, white hierarchy, muted secondary text.

## Typography
Hero identity: 32–40px. Section headings: 18–22px. Labels: 12–14px uppercase.

## Layout Rules
Hero screens use identity, visual, and stats zones. Decorative backgrounds may use absolute positioning; core layout must use grid or flex.

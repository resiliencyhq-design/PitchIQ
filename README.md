# PitchIQ

PitchIQ is the flagship soccer performance product from **WellTrack Performance**, powered by the **WellTrackIQ** technology platform.

## Brand Architecture

- Parent company: **WellTrack Performance**
- Technology platform: **WellTrackIQ**
- Current product: **PitchIQ**

PitchIQ remains the product display name. Repository names, package names, runtime routes, and application architecture should not be renamed unless explicitly approved.

## Current Product Direction

The Football IQ platform foundation is implemented: native missions, seven module hubs, persistent progression, adaptive recommendations, a 12-week curriculum, and assessment benchmarks.

The current development phase expands the initial seven-mission library into a **21-mission curriculum** across seven learning modules. Each module progresses through **Foundation, Developing, and Advanced** learning experiences while reusing the locked runtime and platform architecture.

Current sequence:

1. Sprints 22.1–22.7 — expand each module to three core missions.
2. Sprint 22.8 — apply progressive difficulty rules.
3. Sprint 22.9 — complete coaching and reflection integration.
4. Sprint 23.0 — extend Academy progression and unlocks.
